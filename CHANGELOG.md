# Changelog

## [2026-05-25] 커뮤니티 & 성능 업데이트

### ✨ 새 기능

#### 💬 팁 댓글
- 배송팁 상세 화면 하단에 댓글 섹션 추가
- Supabase Realtime으로 새 댓글 즉시 반영 (팁별 독립 채널)
- 말풍선 UI — 내 댓글은 오른쪽, 팀원 댓글은 왼쪽
- 상대 시간 표시: 방금 전 / N분 전 / N시간 전 / N일 전
- 본인 또는 admin만 댓글 삭제 가능 (소프트 삭제)
- viewer 역할 · 데모 사용자는 댓글 작성 불가
- 댓글 500자 제한 (DB CHECK + UI maxLength 이중 적용)

#### 👑 댓글왕 배지
- 팁 당 가장 많은 댓글을 단 팀원에게 자동 부여
- 최소 2개 이상 달성 시 자격 부여 (1개 = 배지 없음)
- 동점자 모두 배지 수령
- 댓글왕 말풍선은 황금색 테두리 + 살짝 노란 배경으로 구분
- 실시간으로 댓글 순위 변동 즉시 반영

#### ❤️ 팁 좋아요
- 배송팁에 "도움됐어요" 좋아요 버튼 추가
- 좋아요 수 실시간 표시: "N명이 도움됐어요"
- 빨간 하트 토글 (optimistic UI — 즉시 반응)
- viewer 역할 · 데모 사용자 비활성

#### 📍 팀원 위치 공유
- Supabase Realtime Presence 기반 (별도 DB 테이블 없음)
- 앱 켜진 동안만 위치 공유 — 종료 시 자동 소멸
- 하단 👤 버튼으로 공유 ON/OFF (초록색 = 공유 중)
- ON 시 GPS 추적 자동 시작
- 지도에 팀원 초록 원형 마커 + 이름 라벨 실시간 표시
- viewer 역할 · 데모 사용자 제외

#### @멘션 + 🔔 알림
- 댓글 입력 중 `@이름` 입력 시 팀원 자동완성 드롭다운
- 최대 6명까지 필터링 표시, 클릭하면 이름 자동 삽입
- 댓글 등록 시 멘션된 팀원에게 DB 알림 자동 생성
- 댓글 본문 내 `@이름` 파란색 하이라이트 렌더링
- 우상단 🔔 벨 아이콘 + 빨간 미읽 배지 (9+ 표시)
- 알림 탭 시 목록 BottomSheet 오픈 + 전체 읽음 처리
- 미읽 항목: 보라색 왼쪽 선 + 배경 강조

### ⚡ 성능 개선

- **GPS 자동 구역 탐색 제거**: 현재 위치 켤 때마다 `findNearbyZone()`이 전체 구역을 초당 최대 10회 순회하던 문제 제거. GPS는 이제 위치 표시 + 팀원 위치 공유 역할만 수행
